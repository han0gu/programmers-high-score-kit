# 가장 먼 노드 문제 풀이

## 🎯 접근 전략

1. BFS로 접근.
2. 현재 노드의 dist + 1과 그와 연결된 노드의 거리(1번 노드부터 해당 노드까지의 거리임) 비교해 전자가 짧으면 갱신
3. 다 돌고 난 후 가장 먼 거리 값을 가진 노드의 개수 세서 제출

풀어놓고 보니 엉성한 부분이 좀 있다. 다익스트라 식 구현이 섞인 느낌?
* BFS를 쓰면서 dist 테이블은 INF로 초기화 함 
* 큐에 append 시점에 방문 처리를 하지 않아버림

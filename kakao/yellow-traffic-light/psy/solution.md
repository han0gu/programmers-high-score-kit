# 문제 풀이

## 🎯 접근 전략
- **Modular 정수론 문제**
- 일정 주기로 점멸하는 불빛 문제로 생각 (노란불 여부가 중요한 것.)

- 시간 $s$에 최초 점등 후 주기 $c$로 반복될 때, 임의의 시간 $t$가 $t \equiv s \pmod c$를 만족하면 해당 시점에 전구는 켜진 상태
- 시작 시각 $s$, 종료 시각 $e$, 주기 $c$일 때, 임의의 시간 $t$에 대하여$$s \le (t \pmod c) \le e$$위 부등식을 만족할 때 전구는 노란불 상태를 유지

- **탐색 범위**: 
  - 시작: 각 신호등 중 가장 늦게 최초의 노란불 켜지는 순간
  - 끝  : 시작 + 각 신호등 주기의 최소공배수
- 최대 탐색 범위: 16 * 17 * 18 * 19 * 20 == 약 23만 (모든 주기가 서로소)

- 탐색 범위 내 임의의 시간 t에 대하여
  - 모든 신호등이 각자의 start <= t % cycle <= end 를 만족할 경우 -> 정답

- 평행선이면 -1이겠지? / 주가는 동일한데 겹치는 구간이 없는 경우?
  - 동주기, 1사이클 내 모두가 겹치는 구간 없음 -> -1

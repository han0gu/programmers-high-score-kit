# `프로그래머스 코딩테스트 고득점 Kit` 스터디
## 💡 공지사항
- 모든 문제는 [프로그래머스 코딩테스트 고득점 Kit](https://school.programmers.co.kr/learn/challenges?tab=algorithm_practice_kit)에서 발췌하였습니다.
- 🔗 [스터디 디스코드 서버](https://discord.gg/88rzmTkJ)


## 🎯 스터디 목표
- 매주 2문제씩 꾸준히 풀 수 있도록 도와주는 약간의 강제성 (정규 과정에 지장이 가지 않는 선에서)
- 다른 사람들과 아이디어를 공유하면서 얻게 되는 학습 효과


## ✍🏻 스터디 운영 방식
- SeSAC 교육 과정에 맞춰 문제 풀이 언어는 `Python`으로 통일
- 매주 월/목 18:00~19:00에 진행
- 스터디 참석 전
  - 각자 일정에 맞게 문제 풀이 진행 (LLM에게 정답 물어보지 않기)
  - 풀이 내용을 규칙에 맞춰 작성
    - 📝 [풀이 내용 정리 템플릿](https://www.notion.so/2fc16042c2ee804aa07ef9b0ba1bc914?pvs=21)
    - 디렉토리 구조 </br>
      <img width="300" height="384" alt="image" src="https://github.com/user-attachments/assets/2b38df8a-43a8-489a-a708-bae4d6a1e75f" />
  - PR 작성 및 Merge
    - branch naming rule: `<category>/<question>/<member>` (e.g. `sort/largest-number/hsw`)
    - 코드 리뷰는 `optional`
- 스터디 진행 시
  - 본인이 작성한 내용을 간단히 설명하고 자유롭게 토론(질의응답) 진행
- 스터디 종료 후
  - 다음 문제 선정

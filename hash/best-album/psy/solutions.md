# 문제 풀이

## 🎯 접근 전략
- cnt dict에 장르별 재생횟수 누적
- id dict에 장르별 (재생횟수, 고유번호) 저장

- 재생횟수 내림차순으로 장르 정렬
- 각 장르 내에서
  - 1곡이면 그대로 answer에 넣기
  - 2곡 이상이면 `sort(key=lambda x: (x[0], -x[1]), reverse=True)`
    - (x[0], -x[1]): 재생 횟수 역순 정렬하되 id는 오름차순 유지하기 위한 트릭(?)
    - 정렬 후, answer에 앞 2개 넣기
---

## ⚠️ Edge Case
-

    "c3": \[["jazz", "pop"], [150, 800], [1, 0]], # 1개 있는 장르만
    "c4": \[["jazz", "pop", "jazz", "pop"], [150, 800, 150, 800], [1, 3, 0, 2]], # 장르 내 재생횟수 동일

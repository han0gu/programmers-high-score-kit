# 문제 풀이

## 🎯 접근 전략

1. 하나씩 뽑아서 다음거에 있는지 확인(이중반복). 있으면 answer = False로 변경 후 종료
   - 해당 접근의 문제점 1. 시간초과 2. 접두어가 아닌 문자열 전체를 비교
2. 정렬 -> 다음거만 비교(반복문 1번 사용) -> 있으면 answer = False 변경 후 종료
   - 정렬 -> 결국엔 사전순 나열 => 다음거와만 비교해도 괜찮음
   - 1가지 반례 존재.. 뭐가 문제인지 지금은 모르겠음. 생각좀 해봐야할 듯

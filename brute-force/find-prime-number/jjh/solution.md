# 소수 찾기 문제 풀이

## 🎯 접근 전략

1. 주어진 numbers를 사용해서 만들 수 있는 모든 숫자 만들기
   - Counter 클래스 + dfs(백트래킹) 사용해서 구현
     - Counter를 사용하면 문자열/정수 등 사용한 객체를 key, 빈도를 value로 갖는 iterable 객체(딕셔너리) 생성
     - dfs를 이용해 한 자리 수부터 n 자리 수까지의 가능한 모든 숫자 배열 생성
       - Counter를 visited 배열 대신에 사용
   - itertools의 순열을 사용해서 풀 수도 있을 것 같은데, 시간초과 이슈(예상)로 미사용
   - 당연히, 순열 빡 구현으로도 풀 수 있을 것 같은데 과거에 구현해봤을 때 너무 헷갈렸던 기억이 있어서 포기

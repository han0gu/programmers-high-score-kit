# 문제 풀이

## 🎯 접근 전략

### `numbers`를 단위 숫자로 분리
- `n`개의 숫자 식별
- 이번 문제: 최대 `7`개의 숫자 식별
- 예시
    - `17` -> `[1,7]`
    - `011` -> `[0,1,1]`
### 각 숫자를 조합하여 만들 수 있는 모든 경우의 수 계산
- 최대 `nP1` + `nP2` + ... + `nPn` 개의 경우의 수
- 이번 문제: `7P1` + `7P2` + ... + `7P7`
- 예시
    - `[1,7]`
        - `2P1`: 1,7
        - `2P2`: 17, 71
    - `[0,1,1]`
        - `3P1`: 0,1,1
        - `3P2`: 01,10,01,10,11,11
        - `3P3`: 011,101,110
### `Set`을 이용해서 중복 숫자를 제거
- 예시
    - `[1,7]`
        - `2P1`: 1,7
        - `2P2`: 17, 71
    - `[0,1,1]`
        - `3P1`: 0,1
        - `3P2`: 01,10,11
        - `3P3`: 011,101,110


### 모든 경우의 수에 대해 소수 여부를 판단
- `is_prime_number`
